Используя API GroupDocs.Metadata, пользователи могут получить метаинформацию документа, которая включает в себя:

- Формат файла (определяется внутренней структурой)
- Расширение файла
- MIME-тип
- Количество страниц
- Размер файла
- Значение, указывающее, зашифрован ли файл
