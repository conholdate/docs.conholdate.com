## **在 C# 中将 XLS XLSX 转换为 Word **

- 使用 Aspose.Cells 加载 Excel 文件
- 将加载的文件保存为 PDF 流
- 将 PDF 流加载到 Aspose.PDF
- 将 PDF 保存为 DOC 或 DOCX
